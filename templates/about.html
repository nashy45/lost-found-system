About This System

Learn more about the School Lost & Found Management System

Project Overview

This School Lost & Found Management System is a final-year project designed to help educational institutions efficiently track and manage lost and found items. The system provides a user-friendly interface for students, teachers, and staff to report lost or found items, browse current listings, and manage their personal reports.

Key Features

Easy Reporting

Simple form-based reporting system with image upload support

User Dashboard

Personal dashboard to track your reports and view statistics

Admin Management

Comprehensive admin panel for managing all items and users

Search & Browse

Browse all items with filtering and search capabilities

Technology Stack

Backend: Python Flask
Database: SQLite
Frontend: HTML5, CSS3, JavaScript
Authentication: Flask Sessions with password hashing
